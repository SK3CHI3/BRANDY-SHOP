> **The premier e-commerce platform connecting talented Kenyan artists with global customers through authentic, custom-designed merchandise.**

Brandy Shop is more than just an online marketplace—it's a cultural bridge that transforms local artistic talent into sustainable income while sharing Kenya's rich creative heritage with the world. Built with cutting-edge technology and designed for scalability, it represents the future of creative commerce in Africa.

- **📊 Analytics Dashboard** - Track sales, earnings, and customer engagement
- **🚀 Easy Onboarding** - Start selling in minutes with intuitive tools
- **🎯 Marketing Support** - Featured artist programs and promotional opportunities

### For Customers 🛍️
- **🇰🇪 Authentic Kenyan Art** - Unique designs from local talented artists
- **🎨 Custom Products** - T-shirts, hoodies, mugs, bags, phone cases, and more
- **💳 Secure Payments** - InstaPay integration for seamless transactions
- **📦 Order Tracking** - Real-time updates from creation to delivery
- **❤️ Wishlist & Favorites** - Save and organize your favorite designs

### For Culture 🏛️
- **🎭 Heritage Preservation** - Showcase traditional Kenyan patterns and motifs
- **🤝 Community Building** - Connect artists, customers, and culture enthusiasts
- **📚 Cultural Education** - Learn about the stories behind each design
- **🌱 Economic Empowerment** - Support local creative economy

## 🛠️ Tech Stack

### Frontend
- **⚛️ React 18** - Modern UI library with hooks and concurrent features
- **🔷 TypeScript** - Type-safe development for better code quality

- **🧩 Shadcn/ui** - Beautiful, accessible component library

- **📊 Row Level Security** - Database-level security policies

- **📦 npm** - Package management and dependency handling

- **🔧 ESLint & Prettier** - Code quality and formatting

### Integrations
- **💳 InstaPay API** - Local payment processing for Kenya
- **📱 M-Pesa Integration** - Mobile money transactions
- **📧 Email Services** - Transactional emails and notifications
- **📦 Shipping APIs** - Order fulfillment and tracking

### Architecture Highlights
- **🏗️ Component-based architecture** - Reusable, maintainable UI components
- **🔒 Type-safe development** - End-to-end TypeScript implementation
- **📱 Responsive design** - Mobile-first, cross-device compatibility
- **⚡ Performance optimized** - Code splitting, lazy loading, caching
- **🔐 Security-first** - Authentication, authorization, data protection

## 💰 Business Model & Revenue Streams

### Primary Revenue Sources
| Revenue Stream | Description | Rate/Fee |
|----------------|-------------|----------|
| **Artist Commission** | Platform fee from each sale | 15% of product price |
| **Transaction Fees** | Payment processing markup | 2.5% + KSh 10 per transaction |
| **Premium Listings** | Featured artist placements | KSh 2,000/month |
| **Custom Design Fees** | Bespoke artwork commissions | 25% of custom order value |
| **Shipping Markup** | Delivery service margins | 10-15% of shipping cost |

### Market Opportunity
- **🎯 Target Market**: 45M+ Kenyans, 1.3B+ Africans, Global diaspora
- **💹 Market Size**: $2.8B African e-commerce market (growing 18% annually)
- **🎨 Artist Pool**: 50,000+ creative professionals in Kenya
- **📈 Growth Potential**: Cultural products export market expanding globally

## 🚀 Quick Start

### For Developers
```bash
# Clone the repository
git clone https://github.com/SK3CHI3/brandy-shop.git
cd brandy-shop

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your Supabase credentials

# Start development server
npm run dev
```

### For Artists
1. **Sign Up** as an artist at [brandy-shop.vercel.app](https://brandy-shop.vercel.app)
2. **Complete your profile** with portfolio and skills
3. **Upload your designs** and set pricing
4. **Start earning** from your creativity!

### For Customers
1. **Browse** authentic Kenyan designs
2. **Customize** products to your preference
3. **Secure checkout** with InstaPay or card
4. **Track your order** from creation to delivery
